# Diagnostics policy

This repo should only contain redacted summaries and scripts.

Do not commit:

- raw `TS18_diagnostics.zip` bundles;
- copied APKs;
- firmware images;
- serial numbers;
- full logcat dumps containing personal data;
- proprietary binaries.

Commit only summaries such as `redacted/ts18_device_profile.json` and manually curated evidence reports.
