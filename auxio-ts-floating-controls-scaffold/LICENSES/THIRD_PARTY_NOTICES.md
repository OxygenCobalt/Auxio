## MediaFloat

MediaFloat by Leuconoe is an open-source Android floating media-controls app described as Apache-2.0 licensed.

This scaffold uses MediaFloat only as a behaviour and architecture reference. It does not include copied MediaFloat source code.

If developers later copy any MediaFloat code/resources into Auxio-TS, they must:

1. preserve the Apache-2.0 licence text and notices;
2. identify copied/modified files clearly;
3. keep attribution in Auxio-TS notices;
4. verify licence compatibility with Auxio-TS before merging.
