# TS18 Acceptance Checklist

Use this on the actual CR-V TS18.

- [ ] Overlay permission can be granted.
- [ ] Overlay can be enabled from Auxio settings.
- [ ] Overlay is visible over selected launcher.
- [ ] Overlay is visible over navigation.
- [ ] Overlay is visible over stock radio if Android allows it.
- [ ] Overlay does not show over Auxio-TS itself.
- [ ] Previous works.
- [ ] Play/pause works.
- [ ] Next works.
- [ ] Auxio launch button works.
- [ ] Dragging works without interfering with normal taps.
- [ ] Saved position survives app restart.
- [ ] Saved position survives reboot/ACC cycle.
- [ ] No duplicate overlays appear.
- [ ] Disable setting fully removes the overlay.
- [ ] Reverse camera remains usable.
- [ ] CarPlay remains usable.
- [ ] Bluetooth call/audio remains usable.
- [ ] Steering-wheel/media keys are not regressed.
- [ ] Stock TS18 launcher/radio apps are not broken.
